# PySide6 Starter App

Minimal PySide6 application scaffold.

Quick start (Windows PowerShell):

```powershell
python -m venv .venv
.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python -m src.main
```

Files:
- `src/main.py` — runnable example
- `requirements.txt` — dependencies

If you want a packaged app or additional features, tell me what you need.
